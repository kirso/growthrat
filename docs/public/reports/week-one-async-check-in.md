## Experiment Launched

### GrowthRat Week-One Distribution Test

Hypothesis:
One flagship artifact distributed through a primary page plus derivative surfaces should create more early references and stronger engagement than a single long-form page alone.

Assets launched:

- technical flagship page
- growth flagship page
- proof-pack page
- operator replay page
- tracking links for the reference architecture, Test Store/readiness, and
  Charts bridge variants

Primary success signals:

- meaningful interactions driven by the flagship assets
- references to the guides in follow-up replies or discussions
- early clicks into the proof-pack and demo surfaces

## Metrics To Review

This is still a week-one public run, so the review focuses on setup quality,
tracking quality, and early signal quality.

### Distribution metrics
